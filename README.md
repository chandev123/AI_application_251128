## 시작 가이드 (Getting Started)
1. **Clone Repository**
   ```bash
   git clone https://github.com/chandev123/AI_application_251128.git
   ```
2. **Environment Setup**
   - Python 3.8+
   - PyTorch, Torchvision
   - Matplotlib, NumPy, Jupyter Notebook
3. **Run Notebooks**
   - Jupyter Notebook 또는 VS Code에서 `.ipynb` 파일을 열어 실행합니다.
